# meetings

Папка зарезервирована под будущую доменную логику «встреч» (выделение сущности/сервисов поверх календарных событий).

Сейчас реальная работа с заметками встреч живёт в `src/calendar/eventNoteService.ts`. Ранее отдельный `meetingNoteService` был удалён в ходе упрощения структуры, но директория оставлена как место для развития:

- связывание встреч с людьми/проектами;
- индексация/поиск сущностей;
- расширенная модель встречи (не только календарный `CalendarEvent`).
